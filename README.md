# Retail-Sales-Analysis
Excel-based analysis of retail sales data including pivot tables, charts, and an interactive dashboard.
# Project Overview
This project analyzes a retail sales dataset using Microsoft Excel.
The objective is to explore sales performance and customer purchasing behaviour using pivot tables and interactive dashboard.
# Dataset
The dataset was downloaded from Kaggle (cleaned version): https://www.kaggle.com/datasets/hghdhygf/cleaned-retail-sales-dataset?resource=download

It contains transaction-level data with the following key fields:
- Date
- Product Category
- Quantity
- Price per unit
- Total amount
Most transactions are from the year 2023, with only 2 records from January 2024.
# Data Preparation
- Used the cleaned dataset provided on Kaggle
- Created a new column: **Age Group** (18-24, 25-34, 35-44, 45-54, 55+)
- Converted the data into an Excel table
- Prepared the data for pivot table analysis
# Analysis
Pivot tables and chart were created for:
1. Revenue by Product Category  
2. Revenue by Month  
3. Revenue by Age Group  
4. Revenue by Gender  
5. Total Quantity Sold by Product Category  
6. Average Quantity per Transaction by Product Category
# Key Insights
- Electornics generates the highest total revenue among the product categories.
- May records the highest monthly revenue in the dataset.  
- Customers aged 45–54 contribute the highest revenue among all age groups. 
- Female customers generate slightly more revenue than male customers.  
- Clothing has the highest total quantity sold among the categories.  
- On average, customers purchase about 2–3 items per transaction.

Note:  
Since the dataset mainly contains transactions from 2023 (with only 2 records from January 2024), the monthly revenue trend primarily reflects 2023 sales performance and is not suitable for year-over-year comparison.
# Dashboard
An interactive dashboard was created in Excel to visualize:
- Revenue by month  
- Revenue by product category  
- Revenue by age group  
- Revenue by gender  
- Quantity sold by product category

Slicers were added to allow filtering by:
- Product Category  
- Gender  
- Age Group  
- Month
# Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  
# Limitations
- The dataset contains mostly 2023 data with very limited records from 2024.  
- No information is available on promotions, income level, or store location, which limits deeper business analysis.
# Conclusion
This project demonstrates the use of Excel for basic data analysis, pivot tables, and dashboard creation.  
It showcases foundational analytical skills suitable for entry-level data analyst or research assistant roles.
